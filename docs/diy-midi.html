本业务是 人工耳扒 ，将一首歌曲主旋律利用宿主软件人工制作成单轨MIDI，可供UTAU、VOCALOID、Synthesizer V、Ace studio、网易X Studio、DiffSinger、袅袅、Sharpkey、DeepVocal等虚拟歌姬制作软件通用的 .mid 文件。

比起一些软件提供的Ai识别生成MIDI，人工耳扒在音高上更准确、不会跑调、不会突然高8度等问题

人工 VS AI识别 ​

比起一些软件提供的Ai识别生成MIDI，人工耳扒在音高上更准确、不会跑调、不会突然高8度等问题

人工耳扒MIDIAI识别生成MIDI
✅ 音高更准确：绝对音准，助理调教❌音高容易识别错误（识别成和声、高低八度等）
✅ 节奏更准确：人工对节拍，让你的调教不拖拍❌节奏混乱、填词时痛苦折磨
✅ 提前对轨：midi、伴奏对轨，减轻或避免混音对轨工作❌midi伴奏节奏混乱，对不上
✅ 更高质量的伴奏：使用最新最好的算法提取或寻找原版伴奏交付❌效果不好、人声清除不干净

工作流程 ​

  1. QQ/微信联系，发送原曲试听
  2. 确认需求：单人声轨、和声轨、节奏轨、间奏轨、乐器轨（明显的）
  3. 报价
  4. 制作
  5. 交付

交付内容 ​

  1. 原曲： {歌曲名}.wav
  2. MIDI文件： {扒谱日期} {BPM} 胡萝卜须儿 扒谱 {歌曲名}.mid
  3. 分离的原曲伴奏文件：1_{歌曲名}_(Instrumental).wav
  4. 分离的原曲人声文件：1_{歌曲名}_(Vocals).wav

定价规则 ​

最终报价=基础价格+轨道+难度价格

  1. 基础价格：￥25 即单人声轨（短于1min收取半价）
  2. 轨道价格：￥5~￥20 和声轨、节奏轨、间奏轨、乐器轨（明显的）酌情添加
  3. 歌曲难度价格：￥5~￥10 根据歌曲的难度，例如快歌、rap、全曲人声占比等酌情加价
